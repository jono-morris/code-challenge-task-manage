Java Coding Challenge - Task Management API Suite


The application uses an in-memory (H2) database which gets populated with data on startup. The database is available via the h2 console, where it is possible to view database table using the jdbc:h2:mem:testdb JDBC URL.

Functionality

You can achieve the following actions:

  • Fetch all tasks - Select Find Tasks on the home screen then Fetch all Tasks. If the API returns a single task the details of this task will be displayed. If no tasks are returned the application remains on the Find Tasks page.
  • Fetch all overdue tasks - Select Find Tasks on the home screen then Fetch overdue. If the API returns a single task the details of this task will be displayed. If no tasks are returned the application remains on the Find Tasks page.
  • Fetch data for a single task - First select all or all overdue tasks then click a Task's link under the Title column.
  • Add a new task - Select Find Tasks on the home screen then Add Task to enter the Task's details. Any validation errors will be displayed when attempting to add a task.
  • Modify a task - First fetch data for a single task then select Edit Task. Any validation errors will be displayed when attempting to modify the task.
  • Delete a task - First fetch data for a single task then select Delete Task.
